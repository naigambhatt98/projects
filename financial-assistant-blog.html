Building a Personal AI Financial Assistant

Design. Iteration. Generative AI in Practice.

Why I Chose This Project

I’ve always been drawn to practical applications of AI. When given the chance to create something that blended utility with creativity, I focused on a problem people face every day. Personal finance felt like the right space to explore.

My goal was to use generative AI not just for rapid prototyping but to guide thoughtful experience design. Canva’s site builder enabled quick iterations and helped shape a user-focused flow.

Visuals & Process

Using Canva’s generative AI features, I created an interactive prototype for a personal financial assistant. Rather than starting with raw code, I mapped the user journey visually. Explore the assistant site.

Design Thinking and Generative AI

This wasn’t about building another webpage—it was about building trust. What does someone need when overwhelmed with finances? Clarity, simplicity, and a sense of control. With AI-assisted tools, I rapidly prototyped friendly and supportive user flows.

"Generative AI let me focus on intent over interface. I could act as both a designer and a decision-maker."

What I Learned

This project reinforced that the best AI tools don’t replace thought—they accelerate it. Canva reduced busywork and created space for real UX thinking. I also came to value modular design—each tool (budgeting, saving, investing) became a clean component in the system.

Conclusion

This prototype is a small glimpse into what’s possible when thoughtful design meets AI. If it helps reduce someone’s anxiety around money, it has done its job.
